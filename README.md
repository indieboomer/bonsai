# bonsai
A native Windows desktop prototype of a small 3D voxel bonsai tree that grows over time.The tree must be generated procedurally using an L-system-inspired branching algorithm and rendered from small voxel cubes.

This prototype is intentionally narrow in scope.

The purpose of this prototype is to prove that the following core idea works:

procedurally generate a bonsai-like tree
render it as a voxel structure
animate its growth over time
run it as a desktop Windows application in a normal window
keep the code modular so it can later be extended

Must have
Windows desktop app
3D scene
one procedurally generated bonsai tree
L-system-based or L-system-inspired tree skeleton generation
voxel-based branch rendering
animated growth over time
deterministic generation via seed
minimal controls for regenerate / replay / rotate view (with left click and mouse movement)
